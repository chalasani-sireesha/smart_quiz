# Smart Quiz Web Application

## Overview
Smart Quiz is an online quiz application developed using Python and Django. Users can register, log in, attempt quizzes, view results, track their performance, and download certificates.

## Features
- User Registration & Login
- User Profile
- Quiz Categories
- Search Quiz
- Difficulty Levels
- One Attempt Per Quiz
- Quiz Timer
- Dashboard
- Leaderboard
- Quiz History
- Result Calculation
- Certificate Download
- About & Contact Pages
- Password Reset

## Tech Stack
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Installation

```bash
git clone https://github.com/chalasani-sireesha/smart_quiz.git
cd smart_quiz
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
